# @cael/os-light

Phase-0 S1 package frame for the Cael OS light runtime.

This slice ships:

- dependency-free structured output validators
- user-scoped state-kernel primitives
- receipt and audit record schemas
- per-user append-only ledger memory
- synthetic c05 and c06 fixtures with offline tests

The CLI is a Phase-0 stub. `cael-os-light status` prints a placeholder until the S4 install/init slice wires boot status.
